NetTCP

public class NetTCP : Closeable

Provides an asynchronous IO wrapper around a file descriptor. Fully realized for TCP socket types but can also serve as a base for sockets from other families, such as with NetNamedPipe/AF_UNIX.

All sub-class sockets should be switched to utilize non-blocking IO by calling SocketFileDescriptor.switchToNBIO().
