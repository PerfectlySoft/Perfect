JSONEncode

public class JSONEncode

This class encodes Arrays and Dictionaries into JSON text strings

Top-level values which may be encoded include - Array<JSONValue> - Dictionary<String, JSONValue> - JSONArray - JSONDictionary

A value in any of the container types may consist of - Int - Double - String - Bool - JSONNull - JSONArray - JSONDictionary - Array<JSONValue> - Dictionary<String, JSONValue>
